# Shutdown Timer

A sleek, small, aesthetic and native Windows shutdown timer app built with Tauri, React, and Three.js.

## Features
- **Instant System Commands**: Uses native Rust commands to execute and cancel system shutdowns instantly without any popping command windows.
- **Dynamic 3D Globe**: Built with React Three Fiber, featuring a 3D Earth globe that spins slowly and changes appearance depending on the state of the timer.
- **Live Countdown**: See precisely how much time is left in a clear `HH:MM:SS` format.
- **Modern Borderless UI**: Clean, transparent-background window that is fully resizable and maximizable to fit seamlessly on your desktop.

## Usage
Simply pick the number of minutes and press "Start Timer". To abort a scheduled shutdown, click "Cancel".
